# Wikio-Kart

Wikipedia race against your friends and compete for the top spot!

## Setup

Install all the things.

```
$ git clone <repo>
$ cd wikirace/server
$ npm install
$ chmod +x setup.sh
$ ./setup.sh
```

Load the chrome extension locally.

## Running locally

Run the following commands and visit [localhost:3000/](http://localhost:3000/)
```
$ cd server
$ grunt
```
